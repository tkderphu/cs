# What is browser

A browser (web browser) is a software application used to access and view content on the internet.

A browser lets you:

- Visit websites (like google.com or facebook.com)

- View text, images, videos, and animations

- Fill out forms and submit information

- Run web applications (e.g., Gmail, YouTube, online banking)

- Download and upload files
